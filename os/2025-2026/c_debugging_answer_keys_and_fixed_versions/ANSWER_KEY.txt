C DEBUGGING SET — ANSWER KEY
============================

Each original file was intentionally seeded with:
- 3 compilation errors
- 3 warnings
- 2 runtime memory problems suitable for Valgrind

The fixed versions in this folder compile cleanly with:
    gcc -Wall -Wextra -g file.c -o file

Suggested workflow for students:
1. Fix the compile errors first.
2. Rebuild with -Wall -Wextra and remove warnings.
3. Run with Valgrind and fix the memory issues.
